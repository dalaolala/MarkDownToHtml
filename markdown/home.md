#### 实现github的clone以及下载的加速




##### git clone 加速

```
 `本网址`  +  需要clone的内容地址
```

##### 例如
```
git clone  https://github.tothis.pw/https://github.com/nginx/nginx.git
```


##### 下载加速

```
 `本网址`  +  需要下载的内容地址
```

##### 例如
```
https://github.tothis.pw/https://github.com/nginx/nginx/archive/refs/heads/master.zip
```
